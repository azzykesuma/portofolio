About Project

This is a weather prediction app that uses the OpenWeather API to predict the weather in a given location. the map is interactive and can be moved to any point across the globe. there is a search feature also that the user can type any city, but the weather prediction is not yet linked to the search result. there is a three day forecast feature that shows the weather for the next three days.
